Bee datapack

This is a minecraft datapack that lets you turn into bees.

How to use:

Only works in single player at the moment

Download the beepack.zip file and put it directly into the datapacks folder
(requires operator) Use the command /tag @s add canbee to be able to turn into a bee swarm.
To turn into a bee swarm, sneak and jump at the same time. to turn back, jump while on the ground.
You can also use the command /tag @s add canparrot to be able to turn into a parrot, but do not use both commands at once.
